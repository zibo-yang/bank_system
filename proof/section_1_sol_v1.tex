\begin{proof}
Suppose, for the sake of contradiction, that $\sqrt{2}$ is rational. Then there exist integers $a$ and $b$ with $b \neq 0$ such that
\[
\sqrt{2} = \frac{a}{b}.
\]
Among all such representations, choose one with $\gcd(a,b)=1$; that is, assume $\frac{a}{b}$ is written in lowest terms.

Squaring both sides gives
\[
2 = \frac{a^2}{b^2}.
\]
Multiplying both sides by $b^2$ yields
\[
a^2 = 2b^2.
\]
Therefore $a^2$ is even.

We now use the following elementary fact: if $n^2$ is even, then $n$ is even. Indeed, the contrapositive is clear: if $n$ is odd, then $n=2k+1$ for some integer $k$, and hence
\[
n^2 = (2k+1)^2 = 4k^2+4k+1 = 2(2k^2+2k)+1,
\]
which is odd. Thus, since $a^2$ is even, $a$ must be even.

Hence there exists an integer $c$ such that
\[
a = 2c.
\]
Substituting this into $a^2 = 2b^2$, we get
\[
(2c)^2 = 2b^2.
\]
Thus
\[
4c^2 = 2b^2,
\]
and dividing both sides by $2$ gives
\[
2c^2 = b^2.
\]
Therefore $b^2$ is even. By the same elementary fact, $b$ is even.

We have shown that both $a$ and $b$ are even, so $2$ divides both $a$ and $b$. This contradicts the assumption that $\gcd(a,b)=1$.

Therefore our original supposition was false. Hence $\sqrt{2}$ is irrational.
\end{proof}
